## Usage

Import the Badge component and styles.

```jsx
import { Badge } from '@aws-amplify/ui-react';
import '@aws-amplify/ui-react/styles.css';

<Badge>Default badge</Badge>;
```

<Example>
  <Badge>Default badge</Badge>
</Example>

### Badge variations

Use the `variation` prop to change the badge variation. Available options are `default`, `info`, `error`, `warning` and `success`.

```jsx
<Badge>Default</Badge>
<Badge variation="info">Info</Badge>
<Badge variation="error">Error</Badge>
<Badge variation="warning">Warning</Badge>
<Badge variation="success">Success</Badge>
```

<Example>
  <Badge>Default</Badge>
  <Badge variation="info">Info</Badge>
  <Badge variation="error">Error</Badge>
  <Badge variation="warning">Warning</Badge>
  <Badge variation="success">Success</Badge>
</Example>

### Badge sizes

Use the `size` prop to change the badge size. Available options are `small` and `large`.

```jsx
<Badge size="small">Small</Badge>
<Badge>Default</Badge>
<Badge size="large">Large</Badge>
```

<Example>
  <Badge size="small">Small</Badge>
  <Badge>Default</Badge>
  <Badge size="large">Large</Badge>
</Example>

## CSS Styling

### Styling all badges

#### Override Amplify styling using the CSS variables

Using a class and CSS file:

```css
/* styles.css */
:root {
  --amplify-components-badge-background-color: yellow;
}
/* OR */
.my-custom-badge {
  background-color: yellow;
}
```

```jsx
import './styles.css';

<Badge className="my-custom-badge">My badge</Badge>;
```

<Example>
  <style>{`.my-custom-badge { background-color: yellow; } `}</style>
  <Badge className="my-custom-badge">My badge</Badge>
</Example>

#### Override styling using the classname and data attribute selectors

Use the `amplify-badge` class and data attributes to style all badges.

```css
/* Example: Extend all <Badge> styles */
.amplify-badge {
  font-family: 'Times New Roman', Times, serif;
  font-weight: bold;
}

/* Example: Override only info variation styles */
.amplify-badge[data-variation='info'] {
  background-color: rebeccapurple;
}

/* Example: Override size styles */
.amplify-badge[data-size='large'] {
  border: 1px solid black;
}
```

### Styling a badge instance

To override styling on a specific badge

_Using a class selector:_

```css
/* Example: class selector styling override */
.flagged {
  color: white;
  background-color: crimson;
  border-radius: 3px;
}
```

```jsx
<Badge className="flagged">13</Badge>
```

<Example>
  <style>{`.flagged { color: white; background-color: crimson; border-radius: 3px; }`}</style>
  <Badge className="flagged">13</Badge>
</Example>

_Using an id selector:_

```css
/* Example: id selector styling override */
#subscribers {
  color: blue;
  background-color: white;
  font-weight: 900;
  border: 1px solid blue;
}
```

```jsx
<Badge id="subscribers">123K</Badge>
```

<Example>
  <style>{`#subscribers { color: blue; background-color: white; font-weight: 900; border: 1px solid blue; }`}</style>
  <Badge id="subscribers">123K</Badge>
</Example>
