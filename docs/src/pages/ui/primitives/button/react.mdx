## Usage

Import the button component and styles.

```jsx
import { Button } from '@aws-amplify/ui-react';
import '@aws-amplify/ui-react/styles.css';

<Button>Hello world</Button>;
```

<Example>
  <Button>Hello world</Button>
</Example>

### Button sizes

Use the `size` prop to change the button size. Available options are `large` and `small`.

```jsx
import { Button } from "@aws-amplify/ui-react";
import "@aws-amplify/ui-react/styles.css";

<Button size="large">Large</Button>
<Button>Default</Button>
<Button size="small">Small</Button>
```

<Example>
  <Button size="large">Large</Button>
  <Button>Default</Button>
  <Button size="small">Small</Button>
</Example>

### Button variations

Use the `variation` prop to change the button variation. Available options are `primary`, `link`, and none (default).

```jsx
import { Button } from "@aws-amplify/ui-react";
import "@aws-amplify/ui-react/styles.css";

<Button variation="primary">Primary</Button>
<Button>Default</Button>
<Button variation="link">Link</Button>
```

<Example>
  <Button variation="primary">Primary</Button>
  <Button>Default</Button>
  <Button variation="link">Link</Button>
</Example>

## CSS Styling

### Styling all buttons

#### Override Amplify styling using the CSS variables

Using a class and CSS file:

```css
/* styles.css */
:root {
  --amplify-components-button-primary-background-color: #0057ff;
  --amplify-components-button-primary-hover-background-color: #4d89fc;
}
/* OR */
.my-custom-button {
  background-color: #0057ff;
}
.my-custom-button:hover {
  background-color: #4d89fc;
}
```

```jsx
import { Button } from '@aws-amplify/ui-react';
import '@aws-amplify/ui-react/styles.css';
import './styles.css';

<Button className="my-custom-button" variation="primary">
  My button
</Button>;
```

<Example>
  <Button className="my-custom-button" variation="primary">
    My button
  </Button>
</Example>

#### Override styling using the classname and data attribute selectors

Use the `amplify-button` class and data attributes to style all buttons.

```css
/* Example: Extend all <Button> styles */
.amplify-button {
  font-family: 'My Custom Font', 'Comic Sans MS', sans-serif;
  font-weight: bold;
}

/* Example: Override only primary variation styles */
.amplify-button[data-variation='primary'] {
  background-color: teal;
  color: white;
}

/* Example: Override loading styles */
.amplify-button[data-loading='true'] {
  opacity: 0.8;
}

/* Example: Override disabled styles */
.amplify-button[disabled='true'] {
  opacity: 0.8;
}
```

### Styling a button instance

To override styling on a specific button

_Using a class selector:_

```css
/* Example: class selector styling override */
.colorful-button {
  background: linear-gradient(90deg, #fdbb2d 0%, #22c1c3 100%);
}
```

```jsx
<Button className="colorful-button">I'm colorful!</Button>
```

<Example>
  <Button className="colorful-button">I'm colorful!</Button>
</Example>

_Using an id selector:_

```css
/* Example: id selector styling override */
#gradient-button {
  color: white;
  border-radius: 2rem;
  background: rgb(2, 0, 36);
  background: linear-gradient(
    90deg,
    rgba(2, 0, 36, 1) 0%,
    rgba(9, 9, 121, 1) 35%,
    rgba(0, 212, 255, 1) 100%
  );
}
```

```jsx
<Button id="gradient-button">I'm beautiful!</Button>
```

<Example>
  <Button id="gradient-button">I'm beautiful!</Button>
</Example>
